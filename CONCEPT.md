# 캠핑용품 추천 앱 (가칭: GearStar ⭐)

> 캠핑용품을 **수치 기반 별점**으로 객관 비교하고, 구매 링크로 연결해 **제휴 수익(어필리에이트)**을 얻는 큐레이션 앱

작성일: 2026-06-05

---

## 1. 한 줄 정의
가격·무게·패킹사이즈·편의성처럼 **숫자로 비교 가능한 스펙**을 별 5개 척도로 환산해
"어떤 캠핑용품이 내 상황에 맞는지" 한눈에 보여주는 비교·추천 서비스.

## 2. 풀려는 문제
- 캠핑용품은 같은 카테고리(예: 텐트, 침낭, 코펠)라도 가격대가 천차만별.
- 스펙이 제각각 표기돼 있어 **직접 비교가 어렵다** (g vs kg, L vs cm, 모호한 마케팅 문구).
- 초보 캠퍼는 "내 예산/스타일에 뭐가 맞는지" 판단 기준이 없다.

## 3. 핵심 컨셉 — 수치를 별점으로 환산
비교 가능한 정량 지표를 카테고리별 기준으로 정규화해 ⭐1~5로 표시한다.

| 지표 | 설명 | 별점 방향 |
|------|------|-----------|
| 가격(가성비) | 동일 카테고리 가격 분포 대비 | 쌀수록 ⭐↑ |
| 무게 | 백패킹/오토캠핑 모드별 가중 | 가벼울수록 ⭐↑ |
| 패킹 사이즈 | 수납 시 부피 | 작을수록 ⭐↑ |
| 편의성 | 설치 난이도, 사용 편의 (정성→정량 환산) | 편할수록 ⭐↑ |
| 내구성 | 소재/방수지수/리뷰 기반 | 튼튼할수록 ⭐↑ |

> **종합 별점**은 사용자가 고른 "캠핑 스타일"(백패킹 / 오토캠핑 / 미니멀 / 가성비)에 따라
> 각 지표 가중치를 바꿔 재계산한다. → 같은 제품도 사용자별로 추천 점수가 달라짐.

## 4. 화면 구성 (MVP)
1. **홈 / 카테고리** — 텐트, 침낭, 매트, 코펠, 랜턴, 의자/테이블 …
2. **비교 리스트** — 별점 레이더/막대 + 정렬·필터(가격대, 무게, 스타일)
3. **제품 상세** — 지표별 별점, 스펙 원본값, 장단점, **구매 링크(어필리에이트)**
4. **나의 스타일 설정** — 가중치 프로파일 선택 → 추천 점수 개인화

## 5. 수익 모델
- **제휴 마케팅(어필리에이트)**: 쿠팡 파트너스 / 네이버 / 아마존 어소시에이트 등.
- 사용자가 링크 클릭 후 구매 시 수수료 수취 (유튜브 "쿠팡 광고" 방식과 동일).
- 추가 가능: 브랜드 스폰서 큐레이션(광고 표기 필수), 프리미엄 비교 필터.

> ⚠️ **법적 주의**: 어필리에이트 링크는 "이 게시물은 제휴 수수료를 받을 수 있습니다" 고지 의무.
> 쿠팡 파트너스 등은 약관상 고지 문구·심사 요건이 있으니 가입 단계에서 확인.

## 6. 데이터 확보 방안
- 1차: 수작업 큐레이션(인기 제품 50~100개)으로 신뢰도 있는 시드 구축.
- 2차: 오픈마켓 상품 API / 제조사 스펙시트 / 리뷰 크롤링으로 자동화.
- 별점 산식은 **공개·일관**되게 유지해야 신뢰가 생긴다 (블랙박스 추천 지양).

## 7. 차별화 포인트
- 단순 "최저가"가 아니라 **상황 맞춤 다축 비교**.
- 별점 산식 투명 공개 → 신뢰.
- "이 가격대에서 최선" 같은 **예산 구간별 베스트** 제시.

## 8. 다음 액션 (제안)

- [ ] 별점 정규화 산식 v0 정의 (지표별 min-max 또는 분위수)
- [ ] 종이/Figma 와이어프레임 → 비교 리스트 화면 우선
- [ ] 쿠팡 파트너스 가입 요건·고지 의무 리서치
- [ ] MVP 기술 스택 결정 (예: 모바일 우선 → React Native / Flutter, 웹 → Next.js)
