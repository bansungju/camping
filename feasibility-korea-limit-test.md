# 한계 테스트: 국내 브랜드 텐트 데이터 정확도 (2026-06-07)

**방법**: 인기 브랜드 6개 모델을 검색 → 일부는 공식 페이지로 교차검증.

## 핵심 발견 — 실패 패턴 4가지 (이게 진짜 결과물)

### 1. ⛔ 모델명 환각 (가장 위험)
"코바돔(Cova Dome)"은 내가 만들어낸 이름. 실제 헬리녹스 2인 텐트는 **Alpine Dome 2P / Nona Dome**.
→ **정확한 모델명을 모르면 AI가 그럴듯한 이름을 지어낼 수 있다.** 데이터 신뢰의 1순위 리스크.

### 2. 🔴 카테고리 불일치
"코베아 고스트 2인용"은 없음(고스트는 4인+ 패밀리 터널텐트).
어메니티돔(5인 8kg)·잭쉘터(쉘터)·엘찰텐(백패킹 1.8kg)을 한 표에 섞으면 **비교 자체가 무의미.**
→ 비교 전에 **용도(백패킹/오토캠핑/쉘터)와 인원으로 먼저 분류**가 필수.

### 3. 🟡 연식·버전 혼동
제로그램 엘찰텐은 "올뉴/프로/제로본", 연식(21·22)마다 무게가 다름(2.06→1.86kg).
→ **모델명+연식+버전**까지 키로 잡아야 정확. 안 그러면 옛날 스펙과 섞임.

### 4. ⚙️ 공식 페이지 자동 크롤링 실패
제로그램 공식 상품페이지를 직접 긁었더니 **스펙이 텍스트로 안 잡힘**(이미지/JS 안에 박혀 있음).
→ 국내 쇼핑몰·브랜드몰은 **상세스펙을 이미지로** 넣는 경우가 많아 자동 텍스트 추출이 잘 안 됨.

## 정확도 천장 (결론)

- **정확한 모델명 + 메이저 모델**일 때 → 무게·수납·가격이 여러 판매처에서 **일치, 신뢰도 높음** 🟢
- **모델명이 모호하거나 마이너 모델**일 때 → 환각·버전혼동·카테고리오류 발생 🔴
- **공식페이지 직접 크롤링** → 이미지 스펙 때문에 자주 실패. **검색 스니펫·다나와·리뷰사이트 조합이 더 정확.**

## 그래서 데이터 파이프라인은 이렇게 가야 함

1. **모델 화이트리스트 먼저** — "정확한 모델명+연식" 목록을 사람이 확정 (환각 차단)
2. **카테고리 태깅** — 백패킹/오토캠핑/쉘터 × 인원으로 분류 후에만 비교
3. **출처 우선순위** — ① 다나와 스펙 ② 메이저 판매처(무신사/SSG) ③ 리뷰사이트, 공식몰 이미지스펙은 보조
4. **표기 기준 통일** — 무게=최소무게, 수납=가로×세로×높이로 강제 변환
5. **2개 이상 출처 교차검증** — 값 불일치 시 '확인필요' 플래그

## 한 줄 결론

→ 자동화의 핵심은 크롤링이 아니라 **검증된 모델 화이트리스트.** 이게 곧 이 서비스의 해자(moat).
