# SSOT — auth (로그인·회원관리·세션·권한)

> 이 영역의 **단일 진실원본**. auth 관련 모든 변경은 이 문서의 불변규칙과 정합해야 한다(K.1).
> 이 문서 자체를 바꾸려면 사용자 명시 승인(K.2/H 카테고리). 아직 골격 — 첫 auth 작업이 채운다.

## 불변규칙 (invariants)
- 세션 토큰은 httpOnly + Secure 쿠키로만 전달 (로컬스토리지 금지).
- 비밀번호는 평문 저장 금지 — 해시(+salt)만.
- 권한 검사는 서버에서 강제 (프론트 표시 제어는 보조).
