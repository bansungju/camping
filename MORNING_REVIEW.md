## 🚩 발견된 이슈 (아침 피드백 필요) — 평가항목/분류 점검
리뷰 랭킹이 상식과 어긋나는 지점 = 고칠 곳. 모두 측정값이지만 '분류/타당범위' 문제:

1. **랜턴 밝기 20,000lm** = 중국산 과장표기(실제 랜턴 ~100~1000lm). 사용시간 7,200h도 헤드램프 최저모드값.
   → **brightness 타당상한(예 ≤3000lm)·runtime 상한** 필요.
2. **의자 230g/3,510원** = 등산스틱의자·낚시 모노포드 등 '의자 아닌 것' 혼입(실제 캠핑체어 ~0.8~1.5kg).
   → 관련성 필터 강화 + weight 하한(≥400g).
3. **매트 두께 400mm(40cm)** = 에어침대/자충 매트리스 혼입(실제 패드 ~3~10cm).
   → thickness 상한(예 ≤200mm) 또는 '에어베드' 분리.
4. **타프 무게 123g** = '심실링 보수패치·타프스트링' 부속 혼입.
   → 관련성 필터에 보수/패치/스트링/액세서리 제외.
5. **버너 화력 17,025kcal** = 대형 화구(부르스타/야외 강염) 혼입(백패킹 버너 ~2000~3500kcal).
   → 용도 세그먼트(백패킹/대형) 분리.
6. **백패킹텐트 바닥 7.44㎡**(베스트웨이) = 패밀리/풀텐트 오분류 + 외형바닥.
   → 분류 정밀화 + capacity 미상시 floor 외형 의심.
