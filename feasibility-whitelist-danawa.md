# 재테스트: 화이트리스트 방식 + 다나와 크롤링 (2026-06-07)

이전 한계 테스트에서 나온 4대 실패(환각/카테고리오류/버전혼동/공식몰크롤링실패)를
두 가지 처방으로 해결되는지 검증.

---

## A) 화이트리스트 방식 — 카테고리 고정 + 정확한 모델명

**규칙**: 카테고리를 **'백패킹 2인용'으로 고정**, 정확한 모델명+연식만 사용.

### 결과: 교차검증된 비교표 (백패킹 2P)

| 모델 | 무게(최소) | 설치(이너)크기 | 내수압 | 가격대 | 신뢰도 |
|---|---|---|---|---|---|
| 네이처하이크 클라우드업2 플러스 20D | 1.36 kg | 210×130×105 cm | 2,000 mm | 저가(~15만원대) | 🟢 높음 |
| 제로그램 엘찰텐 프로 2P (22년) | 1.86 kg | 220×130×107 cm | 미표기 | 고가(~45만원대) | 🟢 높음(2출처 일치) |
| 헬리녹스 Alpine Dome 2P | 2.55 kg | 수납 47×18cm | 1,500 mm | 최고가(60만원+) | 🟢 높음 |

→ **환각 0건, 카테고리 0오류, 버전혼동 0건.** 이전 실패가 전부 사라짐.
→ 같은 '백패킹 2P'로 묶으니 **무게·내수압 비교가 의미 있게** 작동.

## B) 다나와 스펙 페이지 크롤링 — 텍스트 추출 가능한가?

이전엔 **제로그램 공식몰 직접 크롤링이 실패**(스펙이 이미지 안에 박힘).
이번엔 다나와 스펙 페이지를 직접 긁어봄.

### 결과: ✅ 성공 (공식몰과 정반대)

| 페이지 | 추출된 것 | 못 뽑은 것 |
|---|---|---|
| 엘찰텐 프로 2P (다나와) | 무게 1.86kg, 설치 220×130×107, 재질 나일론15D 립스탑 | 수납크기, 내수압 |
| 엘 찰텐 2P 구형 (다나와) | 무게 2.06kg, 설치 210×130×100, 내수압 ~3000mm | 수납크기, 재질 |

→ **다나와는 스펙이 텍스트 표로 정리돼 있어 자동 추출 잘 됨.** 공식몰보다 훨씬 유리.

### 단, 다나와의 함정 발견 ⚠️
- **항목 누락 흔함**: '수납크기'는 자주 미표기 → 보조 출처 필요.
- **카테고리 태그 오류**: 엘찰텐 구형이 "3인용"으로, 프로2P가 "원터치 그늘막"으로 잘못 태깅됨.
  → 다나와 태그를 그대로 믿으면 안 됨, **모델명 기준 수동 분류 필요.**
- **단종품 가격 = "0원(가격비교 중지)"** → 가격은 다나와만으론 부족, 실판매처 병행 필수.

## 종합 결론 — 두 처방 다 유효 ✅

| 문제(이전) | 처방 | 효과 |
|---|---|---|
| ⛔ 모델명 환각 | 화이트리스트(정확명+연식) | **해결** |
| 🔴 카테고리 오류 | 용도×인원 사전 분류 | **해결** |
| 🟡 버전 혼동 | 연식 명시 | **해결** |
| ⚙️ 공식몰 크롤링 실패 | 다나와 스펙표 사용 | **해결**(텍스트 추출됨) |

### 남은 약점 (다음 과제)
1. **수납크기**는 다나와에 자주 누락 → 리뷰사이트/공식 스펙시트 보조.
2. **국내 실판매가**가 스펙보다 잡기 어려움(단종·변동·직구). 가격 신뢰도가 데이터 중 최약.
3. **내수압 표기 기준 불일치**(미니멈 vs 일반) → 통일 규칙 필요.

## 데이터 파이프라인 v1 (확정안)
```
1. [사람] 카테고리 정의 + 정확 모델명 화이트리스트 작성  ← 핵심/해자
2. [자동] 다나와 스펙표에서 무게/설치크기/재질/내수압 1차 수집
3. [자동] 부족분(수납크기/실판매가)은 메이저 판매처·리뷰로 보강
4. [자동] 표기 기준 통일(무게=최소, 크기=가로×세로×높이)
5. [검증] 2출처 교차, 불일치 시 '확인필요' 플래그 → [사람] 최종 확정
```

**한 줄**: "다나와로 자동 수집 80%, 사람이 화이트리스트+최종검증 20%" 구조면
**정확하고 비교 가능한 데이터셋이 실제로 만들어진다.** 콘텐츠화 충분히 가능.
